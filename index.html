Wanna Make an API Call?

We'll get data from another end point (External Server) then to our site
A Brief Explanation

When the button (Get Users or Get Posts) is clicked it sends a GET request to this end points `https://jsonplaceholder.typicode.com/users` &`https://jsonplaceholder.typicode.com/posts`. Using the fetch method in Javascript, the data is then sent back to us in json format which is in turn parsed then displayed onto the site
